Enhanced Digital Clock

Features

This enhanced digital clock application provides multiple time management features with a modern UI:

  • Light/Dark Mode: Toggle between light and dark themes
  • Analog Clock: Traditional clock face with smooth moving hands and neon glow effects
  • Timer: Set a countdown timer for specific durations
  • Stopwatch: Measure elapsed time with lap functionality
  • Alarm: Set multiple alarms with custom times and sound
  • Modern UI: Glassmorphism design with gradient backgrounds

How to Use

Clock: The main screen shows the current time with both analog and digital displays.

Timer: Set hours, minutes, and seconds for your countdown. Start, pause, or reset the timer as needed.

Stopwatch: Start timing and record lap times. Useful for workouts or competitions.

Alarm: Add multiple alarms by selecting a time and clicking "Add Alarm". Toggle alarms on/off or delete them as needed.

Technology

This application is built with HTML, CSS, and JavaScript. It uses modern CSS features like Flexbox, CSS variables, backdrop-filter for glassmorphism effects, and smooth transitions.
